A gui utility for uploading packages to pypi. Now supports **linux** too.

## Installations

```bash
pip install up-to-pypi
```

Or install from github repository.

```bash
pip install https://github.com/360modder/up-to-pypi/archive/master.zip
```

## Usage

```bash
$ up-to-pypi
```
